# Workplace-Safety-and-Occupational-Injury-Analysis

## Project Overview
This capstone project analyzes occupational injury and illness data to identify trends across industries, company sizes, and incident categories. The goal is to uncover patterns that can support workplace safety decision-making, risk reduction, and public health analysis.

## Business / Public Health Question
Which industries and workplace characteristics show the highest burden of occupational injury and illness?

## Data Source
The dataset comes from publicly available occupational injury and illness records. Due to file size limits, raw data files may not be included directly in this repository. Datasets are to large to upload to Github, they can be downloaded from this link:
www.osha.gov/Establishment-Specific-Injury-and-Illness-Data

## Methods
- Data cleaning and standardization
- Exploratory data analysis
- Injury and illness rate calculations
- Industry and company-size comparisons
- Visualization of key trends
- Interpretation of workplace safety implications

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook
  
## How to Use
1. Download the OSHA datasets and place them in the project directory.  
2. Run the notebooks in order:
   - Exploratory Data Analysis
   - Feature Engineering & Dataset Construction
   - Modeling & Evaluation
